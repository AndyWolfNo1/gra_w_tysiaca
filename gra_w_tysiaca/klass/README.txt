Dla laika:
Aby uruchomić skrypt należy pobrać i zainstalować GIT z linku:
https://git-scm.com/downloads   <=== pozwala pobierać, dodawać i zarzązać wersjami oraz repozytoriami programów z portalu github
Następnie należy pobrać i zainstalować Pythona z linku:
https://www.python.org/downloads/

Żeby pobrać repozytowium należy zainstalować GIT oraz uruchomić CMD(wiersz poleceń) w pliku do którego chcesz pobrać repozytorium,
oraz użycie komendy ' git clone https://github.com/AndyWolfNo1/1000_py.git'

Po pobraniu repozytorium i instalacji Pythona należy uruchomić CMD i przejscie do folderu z pobranym repozytorium oraz 
uruchomienie programu poleceniem 'python3 test.py'
